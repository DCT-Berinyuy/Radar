# Radar Voice Media Utility

Radar is a cross-platform (Flutter/Rust) utility designed for real-time DSP, including active latency monitoring, noise reduction, and media compression. 

## Features
- **Real-time Pipeline Dashboard**: Monitor buffer status, ASIO latency, and system resources.
- **Dynamic Waveform Visualizer**: Live rendering with smooth animations.
- **Hardware Integrations**: Direct integration via Rust bridge.
